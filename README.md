# Currency Converter

A simple currency converter built with React and Vite.

## Features

- Convert between multiple currencies
- Real-time exchange rates
- Responsive design

## Tech Stack

- **Frontend:** React
- **Build Tool:** Vite
- **Styling:** CSS or your preferred framework

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/shishir085/currencyConverter.git ```

2. Navigate into the project directory:
    ```bash
    cd currencyConverter
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Run the development server:
    ```bash
    npm run dev
    ```

Open your browser and go to http://localhost:5173.

## Usage

1. Select a source and target currency.

2. Enter the amount to convert.

3. View the converted value instantly.
